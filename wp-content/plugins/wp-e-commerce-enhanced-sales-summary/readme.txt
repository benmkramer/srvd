== Description ==

Improved Sales Summary Dashboard widget for WP e-Commerce.

For more information visit: http://www.visser.com.au/wp-ecommerce/

== Installation ==

1. Upload the folder 'wp-e-commerce-sales-summary' to the '/wp-content/plugins/' directory

2. Activate 'Enhanced Sales Summary' through the 'Plugins' menu in WordPress

== Usage ==

1. Open the Dashboard page from the WordPress Administration

Done!
